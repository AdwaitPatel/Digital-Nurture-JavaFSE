# Exercise 1 - Logging using SLF4J

## SLF4J
- SLF4J (Simple Logging Facade for Java) is a logging API.
- It works with logging frameworks like Logback.

## Log Levels
- ERROR - Used for serious errors.
- WARN - Used for warning messages.

## Conclusion
- SLF4J provides a standard logging API.
- Logback is the logging implementation used in this exercise.
