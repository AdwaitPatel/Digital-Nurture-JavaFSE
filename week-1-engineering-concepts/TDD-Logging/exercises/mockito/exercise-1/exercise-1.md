# Mockito Exercise 1 - Mocking and Stubbing

## Mocking
- Mocking creates a fake object for testing.
- It avoids calling real external services.

## Stubbing
- Stubbing defines what a mock should return.
- `when(...).thenReturn(...)` is used for stubbing.

## Conclusion
- Mockito isolates dependencies during testing.
- Tests become faster and more reliable.
