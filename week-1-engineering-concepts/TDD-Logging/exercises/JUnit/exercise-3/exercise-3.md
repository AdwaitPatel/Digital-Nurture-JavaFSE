# Exercise 3 - Assertions in JUnit

## Assertions
Assertions are used to verify whether the expected result matches the actual result.

## Common Assertions
- assertEquals() - Checks if two values are equal.
- assertTrue() - Checks if a condition is true.
- assertFalse() - Checks if a condition is false.
- assertNull() - Checks if an object is null.
- assertNotNull() - Checks if an object is not null.

## Conclusion
- If all assertions pass, the test is successful.
- If any assertion fails, JUnit marks the test as failed.
