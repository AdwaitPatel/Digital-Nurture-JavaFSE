# Exercise 4 - Creating and Configuring a Maven Project

## Maven
- Maven is a build automation and dependency management tool.
- Dependencies are managed through `pom.xml`.

## Spring Dependencies
- Spring Context - Provides IoC and Dependency Injection.
- Spring AOP - Supports Aspect-Oriented Programming.
- Spring WebMVC - Used for building web applications.

## Maven Compiler Plugin
- Compiles the project using Java 1.8.

## Conclusion
- Maven manages project dependencies and build lifecycle.
- Spring libraries are added through `pom.xml`.
