# Exercise 1 - Configuring a Basic Spring Application

## Spring Core
- Spring Core provides Dependency Injection (DI) and Inversion of Control (IoC).
- Beans are managed by the Spring IoC Container.

## Steps
1. Create a Maven project.
2. Add Spring Core dependency.
3. Configure beans in `applicationContext.xml`.
4. Load the Spring context and use the bean.

## Conclusion
- Spring manages object creation and dependency injection.
- Beans are configured in `applicationContext.xml`.
